# 🎨 Logos — Ginástica do Cérebro

Arquivos de identidade visual da marca.

## Logos

| Arquivo                        | Layout              | Descrição                                 |
| ------------------------------ | ------------------- | ----------------------------------------- |
| `logo-vertical-colorida.png`   | Vertical (quadrada) | Símbolo em cima + texto embaixo, colorida |
| `logo-vertical-branca.png`     | Vertical (quadrada) | Símbolo em cima + texto embaixo, branca   |
| `logo-horizontal-colorida.png` | Horizontal          | Símbolo + texto ao lado, colorida         |
| `logo-horizontal-branca.png`   | Horizontal          | Símbolo + texto ao lado, branca           |
| `logo-simbolo.svg`             | Vetorial            | Apenas símbolo GC (escalável)             |

## Uso

- Fundos claros → versão colorida
- Fundos escuros → versão branca
- Vertical (quadrada) → espaços com mais altura, redes sociais, perfil
- Horizontal → cantos de cards, headers, espaços compactos
- Web/código → SVG inline ou referência ao arquivo

## Cores da marca

Definidas em `../brand.css` como variáveis CSS reutilizáveis.
