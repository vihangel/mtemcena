# 🎨 Logos — Empório Lúdico

Arquivos de identidade visual da marca.

## Logos

| Arquivo                    | Tipo       | Descrição                             |
| -------------------------- | ---------- | ------------------------------------- |
| `logo principal.png/.svg`  | Principal  | Logo completa (símbolo + texto)       |
| `logo quadrada.png/.svg`   | Quadrada   | Versão compacta quadrada              |
| `logo secundaria.png/.svg` | Secundária | Variação para aplicações alternativas |
| `icone.png/.svg`           | Ícone      | Apenas o símbolo (favicon, avatar)    |
| `icones.png/.svg`          | Ícones     | Conjunto de ícones da marca           |
| `pattern.png/.svg`         | Pattern    | Padrão gráfico para fundos e texturas |

## Uso

- Logo principal → materiais institucionais, headers, destaque
- Logo quadrada → redes sociais, perfil, espaços 1:1
- Logo secundária → quando a principal não se encaixa
- Ícone → favicons, avatares, espaços compactos
- Pattern → fundos de cards, texturas, elementos decorativos

## Cores da marca

Definidas em `../emporio-brand.css` como variáveis CSS reutilizáveis.
